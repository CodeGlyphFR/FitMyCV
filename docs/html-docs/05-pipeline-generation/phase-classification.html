Phase Classification

Premiere phase du pipeline de generation qui analyse et classifie chaque element du CV source selon sa pertinence pour l'offre d'emploi ciblee.

Objectif

La phase de classification determine pour chaque experience, competence et projet du CV source s'il doit etre :

  • keep : Conserver tel quel (tres pertinent)
  • adapt : Adapter le contenu (partiellement pertinent)
  • remove : Supprimer (non pertinent)
  • move_to_projects : Deplacer vers la section projets

Implementation

// lib/cv-generation/phases/classification.js
import { callOpenAI } from '@/lib/openai-core/client';

export async function classifyExperiences(cvData, jobOffer) {
  const result = await callOpenAI({
    featureName: 'classification',
    messages: [
      { role: 'system', content: CLASSIFICATION_PROMPT },
      { role: 'user', content: JSON.stringify({ cv: cvData, job: jobOffer }) }
    ],
    responseFormat: {
      type: 'json_schema',
      json_schema: classificationSchema
    }
  });

  return result;
}
