Docs / Export

Export CV

Le système d'export permet de générer des fichiers PDF et DOCX à partir des CV JSON stockés. L'export utilise des templates HTML/CSS rendus via Puppeteer.

Coût

1 crédit par export (PDF ou DOCX).

Formats Supportés

PDF

  • Rendu haute qualité
  • Polices embarquées
  • Format A4
  • Multi-pages automatique

DOCX

  • Éditable dans Word
  • Conversion depuis HTML
  • Styles préservés
  • Compatible ATS

Export PDF

PROCESS Génération PDF via Puppeteer
Pipeline
// lib/features/export/pdfExporter.js
import puppeteer from 'puppeteer';

export async function exportToPdf(cvData, templateId) {
  // 1. Charger le template
  const template = await loadTemplate(templateId);

  // 2. Injecter les données
  const html = renderTemplate(template, cvData);

  // 3. Lancer Puppeteer
  const browser = await puppeteer.launch({
    headless: true,
    args: ['--no-sandbox', '--disable-setuid-sandbox']
  });

  const page = await browser.newPage();

  // 4. Charger le HTML
  await page.setContent(html, { waitUntil: 'networkidle0' });

  // 5. Générer le PDF
  const pdfBuffer = await page.pdf({
    format: 'A4',
    printBackground: true,
    margin: {
      top: '20mm',
      bottom: '20mm',
      left: '15mm',
      right: '15mm'
    }
  });

  await browser.close();

  return pdfBuffer;
}
Options PDF
format A4 (210mm x 297mm)
printBackground true (couleurs de fond)
margins 20mm haut/bas, 15mm gauche/droite

Export DOCX

PROCESS Génération DOCX
Pipeline
// lib/features/export/docxExporter.js
import { Document, Packer, Paragraph, TextRun } from 'docx';

export async function exportToDocx(cvData) {
  const doc = new Document({
    sections: [{
      properties: {},
      children: [
        // Header
        new Paragraph({
          children: [
            new TextRun({
              text: `${cvData.header.firstName} ${cvData.header.lastName}`,
              bold: true,
              size: 32
            })
          ]
        }),

        // Contact info
        new Paragraph({
          children: [
            new TextRun({ text: cvData.header.email }),
            new TextRun({ text: ' | ' }),
            new TextRun({ text: cvData.header.phone || '' })
          ]
        }),

        // Summary
        new Paragraph({
          heading: 'Heading2',
          children: [new TextRun({ text: 'Résumé Professionnel', bold: true })]
        }),
        new Paragraph({
          children: [new TextRun({ text: cvData.sections.summary?.content || '' })]
        }),

        // Experiences
        ...renderExperiences(cvData.sections.experiences),

        // Skills
        ...renderSkills(cvData.sections.skills),

        // Education
        ...renderEducation(cvData.sections.education)
      ]
    }]
  });

  return await Packer.toBuffer(doc);
}

API Endpoints

Endpoint Méthode Description
/api/cv/[id]/export/pdf GET Exporter en PDF
/api/cv/[id]/export/docx GET Exporter en DOCX

Exemple d'utilisation

// Client-side
const downloadPdf = async (cvId) => {
  const response = await fetch(`/api/cv/${cvId}/export/pdf?template=modern`);

  if (!response.ok) {
    throw new Error('Export failed');
  }

  const blob = await response.blob();
  const url = URL.createObjectURL(blob);

  // Déclencher le téléchargement
  const a = document.createElement('a');
  a.href = url;
  a.download = `cv-${cvId}.pdf`;
  a.click();

  URL.revokeObjectURL(url);
};

Optimisations ATS

Compatibilité ATS

Les exports sont optimisés pour les systèmes de suivi des candidatures (ATS).

  • Structure sémantique : Balises HTML5 appropriées (h1, h2, section)
  • Texte sélectionnable : Pas d'images pour le texte
  • Polices standards : Arial, Helvetica, sans-serif
  • Pas de tableaux complexes : Layout CSS Grid/Flexbox
  • Ordre logique : Contenu dans l'ordre de lecture
