Export PDF

Generation de fichiers PDF a partir des CV au format JSON.

Technologies

  • Puppeteer : Rendu HTML vers PDF
  • Templates HTML : Mise en page du CV
  • Sharp : Optimisation des images

Implementation

// lib/export/pdf.js
import puppeteer from 'puppeteer';

export async function exportToPdf(cvData, templateId) {
  const browser = await puppeteer.launch({ headless: 'new' });
  const page = await browser.newPage();

  // Generer le HTML du CV
  const html = await renderCvTemplate(cvData, templateId);

  await page.setContent(html, { waitUntil: 'networkidle0' });

  const pdf = await page.pdf({
    format: 'A4',
    printBackground: true,
    margin: { top: '10mm', bottom: '10mm', left: '10mm', right: '10mm' }
  });

  await browser.close();

  return pdf;
}
