Types de Taches

Liste des differents types de taches asynchrones gerees par le système.

Types Disponibles

Type Description Duree moyenne
cv_generation Generation de CV optimise 30-60s
pdf_import Import et extraction PDF 15-30s
cv_translation Traduction de CV 20-40s
batch_generation Generation multiple Variable

Modèle BackgroundTask

model BackgroundTask {
  id          String   @id @default(cuid())
  userId      String
  type        String
  status      String   // pending, running, completed, failed
  progress    Int      @default(0)
  result      Json?
  error       String?
  startedAt   DateTime?
  completedAt DateTime?
  createdAt   DateTime @default(now())
}
