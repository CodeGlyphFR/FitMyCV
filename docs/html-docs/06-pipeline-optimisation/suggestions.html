Suggestions IA

Système de generation de suggestions d'amelioration pour optimiser le CV en fonction de l'offre d'emploi.

Types de Suggestions

Type Description Impact
add_keyword Ajouter un mot-clé manquant +2-5% score
rephrase Reformuler pour mieux matcher +1-3% score
add_metric Ajouter des chiffres/metriques +3-5% score
reorder Reorganiser les elements +1-2% score

Implementation

// lib/cv-optimisation/suggestions.js
export async function generateSuggestions(cv, jobOffer, currentScore) {
  const suggestions = await callOpenAI({
    featureName: 'suggestions',
    messages: [
      { role: 'system', content: SUGGESTIONS_PROMPT },
      { role: 'user', content: JSON.stringify({ cv, jobOffer, currentScore }) }
    ],
    responseFormat: { type: 'json_schema', json_schema: suggestionsSchema }
  });

  return suggestions.map(s => ({
    ...s,
    id: generateId(),
    status: 'pending'
  }));
}
