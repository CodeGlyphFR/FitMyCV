Système de Scoring

Le système de scoring évalue chaque section du CV et génère un score global basé sur des critères pondérés. Utilisé à la fois pour le matching et l'optimisation.

Critères d'Évaluation

Critère Poids Description
Technical Skills Match 30% Correspondance compétences techniques
Experience Relevance 25% Pertinence des expériences
Keywords Coverage 20% Couverture des mots-clés de l'offre
Soft Skills Alignment 10% Alignement soft skills
Education Fit 10% Adéquation formation
ATS Optimization 5% Optimisation pour systèmes ATS

Calcul du Score

IA GPT-4o-mini - Scoring
Formule
Score = Σ (critère_i × poids_i)

Exemple:
- Technical Skills: 85 × 0.30 = 25.5
- Experience: 80 × 0.25 = 20.0
- Keywords: 70 × 0.20 = 14.0
- Soft Skills: 75 × 0.10 = 7.5
- Education: 90 × 0.10 = 9.0
- ATS: 80 × 0.05 = 4.0

Score Total = 80/100

Niveaux de Score

Excellent (85-100)

Très bonne correspondance. Peu d'améliorations nécessaires.

Bon (70-84)

Bonne base. Quelques optimisations recommandées.

Moyen (50-69)

Améliorations significatives possibles.

Faible (<50)

Écart important. Révision majeure conseillée.

Scoring par Section

Technical Skills Scoring

function scoreTechnicalSkills(cvSkills, jobRequirements) {
  const required = jobRequirements.skills.technical;
  const niceToHave = jobRequirements.skills.niceToHave || [];

  let matchedRequired = 0;
  let matchedNice = 0;

  required.forEach(skill => {
    if (cvSkills.includes(skill) || hasEquivalent(cvSkills, skill)) {
      matchedRequired++;
    }
  });

  niceToHave.forEach(skill => {
    if (cvSkills.includes(skill)) {
      matchedNice++;
    }
  });

  // Required = 80% du score, Nice-to-have = 20%
  const requiredScore = (matchedRequired / required.length) * 80;
  const niceScore = niceToHave.length > 0
    ? (matchedNice / niceToHave.length) * 20
    : 20;

  return Math.round(requiredScore + niceScore);
}

Experience Relevance Scoring

function scoreExperienceRelevance(experiences, jobOffer) {
  let totalScore = 0;
  let maxScore = 0;

  experiences.forEach(exp => {
    maxScore += 100;

    // Pertinence du titre
    const titleMatch = compareTitles(exp.title, jobOffer.title);
    totalScore += titleMatch * 30;

    // Pertinence des responsabilités
    const respMatch = compareResponsibilities(
      exp.responsibilities,
      jobOffer.responsibilities
    );
    totalScore += respMatch * 50;

    // Pertinence du secteur
    const sectorMatch = compareSectors(exp.company, jobOffer.company);
    totalScore += sectorMatch * 20;
  });

  return Math.round((totalScore / maxScore) * 100);
}
