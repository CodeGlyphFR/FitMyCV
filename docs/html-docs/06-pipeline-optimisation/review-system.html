Système de Review

Interface permettant a l'utilisateur de valider ou rejeter les suggestions d'optimisation.

Etats des Suggestions

Statut Description
pending En attente de decision
accepted Acceptee, prete a appliquer
rejected Rejetee par l'utilisateur
applied Appliquee au CV

API Endpoints

// POST /api/cv/[id]/suggestions/[suggestionId]/accept
// POST /api/cv/[id]/suggestions/[suggestionId]/reject
