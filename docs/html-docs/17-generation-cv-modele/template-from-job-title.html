CV Modèle depuis Titre de Poste

Génération d'un CV modèle (fictif professionnel) à partir d'un simple titre de poste. L'IA génère un CV complet basé sur les exigences typiques du marché pour ce métier.

Coût

Feature : generate_from_job_title - Utilise les limites mensuelles ou débite 1 crédit. Le crédit est débité au lancement et remboursé en cas d'échec.

Différence avec "Depuis Offre"

Cette méthode génère un CV générique basé sur le titre seul. Pour un CV plus précis et adapté aux exigences spécifiques d'un employeur, utilisez la génération depuis offre.

Mapping automatique

La langue de l'interface utilisateur est automatiquement mappée vers la valeur attendue par l'API. Le mapping se fait via LOCALE_TO_LANGUAGE dans CvGeneratorModal.jsx.

Gestion des Erreurs

Erreur Cause Action
NO_JOB_TITLE_PROVIDED Titre manquant ou vide Rejet immédiat, pas de débit
FEATURE_LIMIT_REACHED Limite mensuelle atteinte Rejet, proposition achat crédits
invalidJsonResponse Réponse GPT non-JSON Remboursement crédits
incompleteCv CV sans champs essentiels Remboursement crédits
gptGenerationFailed Échec GPT Remboursement crédits
Task cancelled Annulation utilisateur Remboursement crédits
