CV Modèle depuis Offre d'Emploi

Génération d'un CV modèle (fictif professionnel) à partir d'une URL ou d'un PDF d'offre d'emploi. L'IA extrait les exigences de l'offre puis génère un CV de candidat idéal.

Coût

Feature : gpt_cv_generation - Utilise les limites mensuelles ou débite 1 crédit. Le crédit est débité au lancement et remboursé en cas d'échec.

Étape 3 : Formatage pour le Prompt

PROCESS Formatage offre
Fonction
// lib/features/template-generation/service.js
function formatJobOfferForTemplate(extraction, source, title) {
  const sections = [];

  sections.push(`Offre d'emploi extraite depuis: ${source}`);
  if (title) sections.push(`Titre: ${title}`);

  if (extraction.company) {
    sections.push(`ENTREPRISE: ${extraction.company}`);
  }

  if (extraction.skills?.required?.length > 0) {
    sections.push(`COMPÉTENCES REQUISES:\n${
      extraction.skills.required.map(s => `- ${s}`).join('\n')
    }`);
  }

  // ... autres sections

  return sections.join('\n\n');
}

Étape 5 : Sauvegarde

OUTPUT CvFile créé
Stockage
// lib/features/template-generation/job.js
const cvFile = await prisma.cvFile.create({
  data: {
    userId,
    filename: `cv-${Date.now()}.json`,
    content: cvContent,        // JSON du CV généré
    language: jobOfferLanguage || 'fr',
    metadata: {
      origin: 'template_from_offer',
      jobOfferId: jobOfferId,
      generatedAt: new Date().toISOString()
    }
  }
});

// Lier le CV à l'offre d'emploi
if (jobOfferId) {
  await prisma.cvFile.update({
    where: { id: cvFile.id },
    data: { jobOfferId }
  });
}

Gestion des Erreurs

Erreur Cause Action
NO_SOURCE_PROVIDED Ni URL ni PDF fourni Rejet immédiat, pas de débit
INVALID_LINKS_FORMAT JSON des liens invalide Rejet immédiat, pas de débit
FEATURE_LIMIT_REACHED Limite mensuelle atteinte Rejet, proposition achat crédits
EXTRACTION_FAILED Échec extraction offre Remboursement crédits
GENERATION_FAILED Échec GPT Remboursement crédits
Task cancelled Annulation utilisateur Remboursement crédits
