Onboarding

Le flux d'onboarding guide les nouveaux utilisateurs à travers la création de leur premier CV, avec tracking des étapes pour analyser les points de friction.

Modèle de Données

model OnboardingProgress {
  id          Int       @id @default(autoincrement())
  userId      String    @unique
  step        Int       @default(1)
  completed   Boolean   @default(false)
  method      String?   // pdf_import, manual
  startedAt   DateTime  @default(now())
  completedAt DateTime?
  metadata    Json?     // Données additionnelles

  user        User      @relation(fields: [userId], references: [id])
}

Étapes Détaillées

Étape Description Action tracking
1 Création du premier CV (Import PDF ou création manuelle) onboarding_cv_created
2 Découverte du mode édition onboarding_edit_mode_discovered
3 Première génération/optimisation onboarding_first_generation
4 Onboarding terminé onboarding_completed

Méthodes de Création CV

Import PDF

  • Upload fichier existant
  • Extraction IA automatique
  • Coût configurable dans Settings

Création Manuelle

  • Formulaire étape par étape
  • Guidage contextuel
  • Gratuit (pas de crédit requis)

Tracking Analytics

// lib/analytics/onboarding.js

export async function trackOnboardingStep(userId, step, metadata = {}) {
  await prisma.onboardingProgress.upsert({
    where: { userId },
    update: {
      step,
      metadata: {
        ...existingMetadata,
        [`step${step}At`]: new Date().toISOString(),
        ...metadata
      }
    },
    create: {
      userId,
      step,
      metadata
    }
  });

  // Envoyer à analytics
  await trackEvent('onboarding_step', {
    userId,
    step,
    ...metadata
  });
}

export async function completeOnboarding(userId) {
  await prisma.onboardingProgress.update({
    where: { userId },
    data: {
      completed: true,
      completedAt: new Date()
    }
  });

  await trackEvent('onboarding_completed', { userId });
}

Métriques Clés

Métrique Description Objectif
Completion Rate % d'utilisateurs qui terminent > 60%
Drop-off par étape Où les utilisateurs abandonnent < 15% par étape
Time to First CV Temps jusqu'au premier CV créé < 10 minutes
Method Distribution Répartition des méthodes choisies -

Dashboard Admin

Le dashboard admin affiche les métriques d'onboarding via OnboardingTab.jsx :

  • Graphique funnel de conversion
  • Taux de complétion par période
  • Distribution des méthodes de création
  • Points de drop-off identifiés
