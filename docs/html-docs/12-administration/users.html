Gestion des Utilisateurs

Interface d'administration pour gérer les comptes utilisateurs.

Fonctionnalites

  • Liste des utilisateurs avec filtres et recherche
  • Details du compte (plan, credits, activite)
  • Actions : suspendre, supprimer, modifier credits
  • Historique des transactions

API Endpoints

Endpoint Description
GET /api/admin/users Liste paginee des utilisateurs
GET /api/admin/users/[id] Details d'un utilisateur
PUT /api/admin/users/[id] Modifier un utilisateur
POST /api/admin/users/[id]/credits Ajouter/retirer des credits
