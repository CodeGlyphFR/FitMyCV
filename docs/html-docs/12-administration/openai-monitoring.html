Monitoring OpenAI

Suivi des couts et de l'utilisation de l'API OpenAI.

Metriques Suivies

  • Tokens utilises (prompt + completion)
  • Cout estime par requête
  • Cout total par jour/mois
  • Repartition par feature

Modèle OpenAIUsageLog

model OpenAIUsageLog {
  id               String   @id @default(cuid())
  userId           String?
  featureName      String
  model            String
  promptTokens     Int
  completionTokens Int
  totalTokens      Int
  estimatedCost    Decimal
  duration         Int      // ms
  createdAt        DateTime @default(now())
}
