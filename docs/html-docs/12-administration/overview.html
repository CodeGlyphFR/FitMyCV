Administration

Le dashboard d'administration permet de gérer tous les aspects de l'application : utilisateurs, abonnements, crédits, monitoring OpenAI, templates email et paramètres système.

Accès restreint

L'accès au dashboard admin nécessite le rôle ADMIN sur le compte utilisateur. Vérifié via session.user.role === 'ADMIN'.

Architecture du Dashboard

Modules disponibles

Module Description API Endpoint
Overview KPIs globaux (MRR, utilisateurs, CVs générés) /api/analytics/*
Utilisateurs Gestion des comptes utilisateurs /api/admin/users
Plans Abonnements Stripe /api/admin/subscription-plans
Crédits Packs et transactions /api/admin/credit-packs
OpenAI Monitoring coûts et alertes /api/admin/openai-*
Email Templates et logs /api/admin/email-*
Settings Configuration système (modèles IA, coûts crédits) /api/admin/settings

KPIs et Métriques

Métriques Financières

Métrique Description Source
MRR Monthly Recurring Revenue Somme des abonnements actifs
ARR Annual Recurring Revenue MRR × 12
ARPU Average Revenue Per User MRR / utilisateurs payants
Churn Rate Taux de désabonnement Annulations / total abonnés

Métriques Usage

Métrique Description
CVs générés Nombre de CV créés via pipeline génération
CVs optimisés Nombre de CV améliorés via pipeline optimisation
PDFs importés Nombre d'imports PDF réussis
Exports Nombre d'exports PDF/DOCX
Coûts OpenAI Coûts par feature, utilisateur, période

Monitoring OpenAI

Le système de monitoring permet de suivre les coûts OpenAI en temps réel et de configurer des alertes.

Types d'Alertes

Type Description Exemple
user_daily Coût par utilisateur par jour Alerte si > 5$ / jour
user_monthly Coût par utilisateur par mois Alerte si > 50$ / mois
global_daily Coût total par jour Alerte si > 100$ / jour
global_monthly Coût total par mois Alerte si > 2000$ / mois
feature_daily Coût par feature par jour Alerte si cv_generation > 50$ / jour

Tracking des Tokens

// Chaque appel OpenAI est tracké
{
  userId: "user_id",
  featureName: "cv_generation",
  model: "gpt-4o",
  promptTokens: 1500,
  cachedTokens: 800,        // Tokens du cache (90% moins cher)
  completionTokens: 500,
  totalTokens: 2000,
  estimatedCost: 0.025,     // En USD
  duration: 2500            // En ms
}

Gestion des Settings

Les paramètres système sont stockés dans la table Setting et peuvent être modifiés en temps réel.

Catégories de Settings

Catégorie Exemples
app maintenance_mode, registration_enabled
subscription subscription_mode_enabled, default_plan_id
credits welcome_credits, credits_gpt_cv_generation
openai model_cv_generation, model_match_score
cv cv_max_versions, cv_encryption_enabled
