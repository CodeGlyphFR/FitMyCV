Import PDF

Le système d'import PDF utilise l'API Vision de GPT-4o pour extraire et structurer les informations d'un CV au format PDF vers le format JSON standardisé.

Coût

5 crédits par import. Le crédit est débité au lancement et remboursé en cas d'échec.

Étape 2 : Conversion en Images

PROCESS PDF → Images
Traitement
// lib/features/pdf-import/converter.js
import { PDFDocument } from 'pdf-lib';
import sharp from 'sharp';

export async function convertPdfToImages(pdfBuffer) {
  const pdfDoc = await PDFDocument.load(pdfBuffer);
  const pages = pdfDoc.getPages();
  const images = [];

  for (let i = 0; i < pages.length; i++) {
    // Render page to PNG
    const pageImage = await renderPage(pages[i]);
    // Resize pour optimiser tokens
    const resized = await sharp(pageImage)
      .resize(1200, null, { fit: 'inside' })
      .png()
      .toBuffer();
    images.push(resized.toString('base64'));
  }

  return images;
}
Output

Array de strings base64 (1 par page)

Prompt Structure
// lib/features/pdf-import/prompts/extract-cv.md

# Mission
Extraire TOUTES les informations du CV en image.

# Règles
1. Extraire uniquement ce qui est visible
2. Ne JAMAIS inventer d'information
3. Conserver la langue originale du CV
4. Structurer selon le schema JSON fourni

# Sections à extraire
- Informations personnelles (header)
- Résumé professionnel
- Expériences professionnelles
- Compétences (techniques, soft, langues)
- Formation
- Projets
- Certifications
- Autres (bénévolat, hobbies...)

Étape 4 : Sauvegarde

OUTPUT CvFile créé
Stockage
const cvFile = await prisma.cvFile.create({
  data: {
    userId,
    filename: `import-${Date.now()}.json`,
    content: extractedCv, // JSON structuré
    isDefault: false,
    metadata: {
      origin: 'pdf_import',
      originalFilename: uploadedFile.name,
      importedAt: new Date().toISOString()
    }
  }
});

// Créer version initiale
await prisma.cvVersion.create({
  data: {
    cvFileId: cvFile.id,
    versionNumber: 1,
    content: extractedCv,
    origin: 'pdf_import',
    description: 'Import depuis PDF'
  }
});

Gestion des Erreurs

Erreur Cause Action
INVALID_FILE_TYPE Fichier non-PDF Rejet immédiat, pas de débit
FILE_TOO_LARGE PDF > 10MB Rejet immédiat, pas de débit
TOO_MANY_PAGES PDF > 5 pages Rejet immédiat, pas de débit
EXTRACTION_FAILED Échec GPT-4o Remboursement crédits
VALIDATION_FAILED JSON invalide Remboursement crédits
