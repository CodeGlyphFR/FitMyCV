API Reference

FitMyCV.io expose 113 endpoints API organisés en domaines fonctionnels. Toutes les API utilisent REST avec JSON.

Base URL

https://app.fitmycv.io/api

Authentification

La plupart des endpoints nécessitent une session authentifiée via NextAuth.js :

// Cookie de session automatique
// Ou header Authorization pour API externe
Authorization: Bearer {session_token}

Format des Réponses

Succès

{
  "data": { ... },
  "meta": {
    "timestamp": "2024-01-15T10:30:00Z"
  }
}

Erreur

{
  "error": "Description de l'erreur",
  "code": "ERROR_CODE",
  "details": { ... },  // Optionnel
  "timestamp": "2024-01-15T10:30:00Z"
}

Codes d'Erreur HTTP

Code Signification Exemple
200 Succès GET, PUT réussi
201 Créé POST réussi
400 Bad Request Validation échouée
401 Non authentifié Session manquante/expirée
403 Accès refusé Pas les droits requis
404 Non trouvé Ressource inexistante
429 Limite atteinte Feature limit / rate limit
500 Erreur serveur Erreur interne

Pagination

Les endpoints de liste supportent la pagination :

GET /api/cv?page=1&limit=20

// Response
{
  "data": [...],
  "meta": {
    "page": 1,
    "limit": 20,
    "total": 45,
    "totalPages": 3
  }
}
