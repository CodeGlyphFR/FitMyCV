Endpoints Extension Navigateur

Endpoints dédiés à l'extension Chrome/Firefox. Authentification par JWT Bearer via withExtensionAuth.

Authentification JWT

Tous les endpoints /api/ext/* nécessitent un header Authorization: Bearer {jwt}. Le JWT est obtenu via /api/auth/extension-token et vérifié par le middleware withExtensionAuth (lib/api/withExtensionAuth.js).

Génération CV (1 endpoint)

MéthodeEndpointDescription
POST/api/ext/background-tasks/generate-cv-from-contentGénérer un CV à partir du contenu Markdown d'une offre d'emploi extraite par l'extension

POST /api/ext/background-tasks/generate-cv-from-content

L'extension envoie le contenu Markdown brut extrait de la page d'offre d'emploi. Le serveur exécute extractJobOfferFromMarkdown() pour structurer les données, puis lance le pipeline de génération de CV.

Request

{
  "cvFileId": "cv_xxx",
  "markdown": "# Offre d'emploi\n## Chef de projet...",
  "url": "https://linkedin.com/jobs/xxx",
  "siteName": "LinkedIn",
  "deviceId": "device_xxx"
}

Response (202 Accepted)

{
  "success": true,
  "queued": true,
  "taskId": "task_ext_xxx"
}

CORS

Les requêtes de l'extension sont gérées par proxy.js (pas middleware.js). L'extension utilise les APIs browser.runtime pour les requêtes cross-origin.
