Endpoints Admin

Endpoints reserves aux administrateurs.

Acces restreint

Ces endpoints necessitent le role admin sur le compte utilisateur.

Utilisateurs

Endpoint Méthode Description
/api/admin/users GET Liste des utilisateurs
/api/admin/users/[id] GET/PUT/DELETE Gestion utilisateur
/api/admin/users/[id]/credits POST Modifier credits

Configuration

Endpoint Méthode Description
/api/admin/plans GET/POST/PUT Gestion plans
/api/admin/settings GET/PUT Parametres système
/api/admin/stats GET Statistiques globales
