Endpoints Authentifiés

Endpoints nécessitant une session utilisateur valide via cookie NextAuth.js. Couvrent la gestion des CV, les tâches background, les abonnements, les crédits, l'export et le compte utilisateur.

Authentification requise

Tous les endpoints ci-dessous vérifient la session via auth() (NextAuth.js). Sans session valide : 401 Unauthorized.

POST /api/background-tasks/create-template-cv

Génère un ou plusieurs CV modèles (fictifs professionnels) à partir d'offres d'emploi.

Request (FormData)

{
  links: JSON.stringify(["https://example.com/offre"]),
  files: [File],                    // PDFs d'offres
  model: "gpt-4o",                  // Optionnel
  deviceId: "device_xxx",           // Requis
  recaptchaToken: "..."             // Optionnel
}

Response (202 Accepted)

{
  "success": true,
  "queued": true,
  "taskIds": ["task_template_link_xxx"],
  "tasksCount": 1
}

POST /api/background-tasks/generate-cv-from-job-title

Génère un CV modèle à partir d'un titre de poste.

}

  "queued": true,

Localisation CGV

Le paramètre locale (fr, en, es, de) affiche le message d'acceptation des CGV dans la langue de l'utilisateur lors du checkout Stripe.
