Génération CV Modèle

Endpoint Méthode Description

POST /api/background-tasks/create-template-cv

Génère un ou plusieurs CV modèles (fictifs professionnels) à partir d'offres d'emploi.

Request (FormData)

{
  links: JSON.stringify(["https://example.com/offre"]), // Array d'URLs
  files: [File],                                         // PDFs d'offres
  model: "gpt-4o",                                       // Optionnel
  deviceId: "device_xxx",                                // Requis
  recaptchaToken: "..."                                  // Optionnel
}

Response (202 Accepted)

{
  "success": true,
  "queued": true,
  "taskIds": ["task_template_link_xxx"],
  "tasksCount": 1
}

Feature

gpt_cv_generation - 1 crédit par génération

POST /api/background-tasks/generate-cv-from-job-title

Génère un CV modèle à partir d'un titre de poste.

Request (FormData)

{
  jobTitle: "Chef de projet IT",  // Requis
  language: "français",          // Défaut: français
  model: "gpt-4o",                // Optionnel
  deviceId: "device_xxx",         // Requis
  recaptchaToken: "..."           // Optionnel
}

Response (202 Accepted)

{
  "success": true,
  "queued": true,
  "taskId": "task_job_title_xxx"
}

Feature

generate_from_job_title - 1 crédit par génération
