Webhooks (1 endpoint)

MéthodeEndpointSourceDescription
POST/api/webhooks/stripeStripeTraite les événements Stripe (checkout, subscription, invoice)
Sécurité Webhook

Le webhook Stripe vérifie la signature via stripe.webhooks.constructEvent() avec le secret STRIPE_WEBHOOK_SECRET. Les requêtes non signées sont rejetées avec 400.
