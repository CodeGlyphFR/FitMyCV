Gestion des CV

FitMyCV.io structure chaque CV en 8 sections standardisees avec un format riche et valide automatiquement. Le cycle de vie complet du document est couvert : creation manuelle, import PDF par IA, versioning intelligent, traduction en 4 langues et export professionnel en PDF ou Word.

Format structure en 8 sections

Chaque CV est stocke dans un format structure et valide automatiquement, bien au-dela d'un simple document texte. Ce format garantit la coherence des donnees a travers toutes les operations : edition, optimisation IA, traduction et export.

Les sections marquees * sont obligatoires.

Section Obligatoire Contenu
En-tete Oui Nom complet, poste actuel, coordonnees (email, telephone, liens, localisation)
Resume professionnel Oui Synthese du profil en quelques lignes
Competences Oui 4 categories (competences techniques, outils, methodologies, qualites personnelles) avec echelle de maitrise 0-5
Experiences Oui Postes avec titre, entreprise, dates, responsabilites, realisations et competences utilisees
Formation Non Diplomes, etablissements et dates
Langues Non Langues parlees avec niveaux CECRL (A1 a C2)
Informations complementaires Non Centres d'interet, certifications, benevolat
Projets Non Projets personnels ou open source avec descriptions et competences utilisees

Competences structurees

Les competences sont classees en 4 categories avec une echelle de maitrise de 0 a 5 pour chaque competence. Cette structuration permet a l'IA de comparer precisement les competences du candidat avec les exigences d'une offre d'emploi.

Categorie Exemples
Competences techniques JavaScript, Python, SQL, Machine Learning, Architecture logicielle
Outils Git, Docker, VS Code, Figma, Jira
Methodologies Agile/Scrum, TDD, CI/CD, Design Thinking
Qualites personnelles Leadership, Communication, Travail en equipe, Resolution de problemes

Personnalisation de l'affichage

L'ordre des sections est entierement personnalisable par l'utilisateur. Les titres des sections sont automatiquement traduits dans la langue du CV (francais, anglais, espagnol, allemand). Le format est valide automatiquement a chaque modification pour garantir la coherence des donnees.

Versioning intelligent

FitMyCV.io maintient un historique des versions de chaque CV, avec une approche selective : seules les modifications effectuees par l'IA (optimisation, adaptation) creent une nouvelle version. Les editions manuelles modifient directement le contenu sans versionner, pour eviter une proliferation inutile de versions.

Principes cles

Aspect Comportement
Creation de version Uniquement lors des optimisations et adaptations IA. Les editions manuelles ne creent pas de version.
Version originale La version initiale du CV (avant toute optimisation) est toujours preservee comme reference.
Restauration prudente L'etat actuel est sauvegarde avant la restauration. Aucune donnee n'est perdue, tout l'historique est preserve.
Restauration destructive Retour direct a une version anterieure avec suppression de toutes les versions plus recentes. Irreversible.
Rotation automatique Maximum 5 versions conservees par CV (configurable). Les plus anciennes sont supprimees automatiquement au-dela de cette limite.
Donnees preservees Chaque version sauvegarde le contenu complet du CV ainsi que les scores d'adequation associes.

Import PDF par vision IA

FitMyCV.io permet d'importer un CV existant au format PDF. L'IA analyse visuellement chaque page du document et reflechit a sa structure avant d'en extraire les informations. Cette approche par vision va bien au-dela d'un simple OCR : l'IA comprend la mise en page, identifie les sections et structure les donnees dans le format standardise.

Detail des etapes

  1. Envoi et validation : Verification du type de fichier, de la taille (10 Mo maximum) et du nombre de pages (10 pages maximum)
  2. Conversion en images : Chaque page du PDF est convertie en image haute resolution, optimisee pour l'analyse visuelle
  3. Analyse visuelle par l'IA : L'IA examine les images et reflechit a haute voix avant d'extraire : elle identifie la structure du document, la langue detectee, les eventuels problemes de lisibilite et sa strategie d'extraction. Le resultat est un format structure garanti valide.
  4. Reconstruction du CV : Les donnees extraites sont normalisees et injectees dans les 8 sections standardisees
  5. Sauvegarde : Le CV est cree avec la source tracee (origine PDF), pret a etre edite ou optimise par l'IA
Garanti sans risque : L'import s'execute en tache de fond, sans bloquer l'interface. En cas d'echec (document illisible, resultat invalide), les credits sont automatiquement rembourses. L'utilisateur ne perd jamais de credits pour un import echoue.

Traduction en 4 langues

FitMyCV.io permet de traduire un CV dans l'une des 4 langues supportees : francais, anglais, espagnol et allemand. La traduction est realisee par l'IA avec des regles intelligentes qui vont au-dela d'une simple traduction mot-a-mot.

Regles de traduction intelligentes

Regle Description
Preservation de la structure Seuls les contenus textuels sont traduits. La structure du CV (8 sections, ordre, competences) reste identique.
Niveaux de competences Les niveaux de maitrise sont traduits avec les equivalences locales (Debutant → Beginner / Principiante / Anfanger).
Codes CECRL preserves Les codes de niveau de langue (A1, B2, C1...) restent inchanges pour la reconnaissance internationale. Seuls les libelles sont traduits.
Titres de sections Les titres de sections sont automatiquement adaptes a la langue cible.
Original intact La traduction cree un nouveau document. Le CV original n'est jamais modifie.
Traduction professionnelle : L'IA produit des traductions fideles et culturellement adaptees, avec une terminologie professionnelle appropriee a chaque langue.

L'utilisateur dispose d'un controle fin sur le contenu exporte :

  • Selection par section : activer ou desactiver chaque section du CV
  • Selection par element : masquer des experiences, competences ou formations individuellement (ideal pour adapter le CV a une offre specifique)
  • Ordre personnalisable : reorganiser les sections pour mettre en avant les informations les plus pertinentes
  • Templates de selection : sauvegarder une configuration de selection pour la reutiliser (ex: "CV developpeur frontend" vs "CV fullstack")

Export multilingue

Les libelles du CV exporte (titres de sections, niveaux de competences, niveaux CECRL) sont automatiquement traduits dans la langue du CV parmi les 4 langues supportees.

Remboursement automatique : Si l'export echoue pour quelque raison que ce soit, le credit est automatiquement restitue. L'utilisateur ne perd jamais de credits pour un service non rendu.
