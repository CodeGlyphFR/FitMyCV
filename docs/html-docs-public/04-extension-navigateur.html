Extension Navigateur

Capturez et adaptez une offre d'emploi en un clic, sans quitter votre navigateur. L'extension FitMyCV pour Chrome et Firefox transforme votre navigation quotidienne en outil de productivité : elle détecte automatiquement les offres d'emploi sur plus de 14 sites, extrait leur contenu et lance la génération d'un CV optimisé directement depuis la page.

Architecture de l'extension

L'extension repose sur l'architecture Manifest V3, le standard actuel imposé par les navigateurs. Elle est construite autour de 5 composants spécialisés qui communiquent entre eux via un système de messages interne.

Compatibilité Chrome et Firefox

L'extension est développée à partir d'une seule base de code avec une compatibilité native Chrome et Firefox. Le système de build produit automatiquement deux versions adaptées à chaque navigateur, sans duplication de code. L'internationalisation couvre 4 langues (français, anglais, espagnol, allemand) avec détection automatique de la langue du navigateur.

Détection intelligente des offres d'emploi

L'extension utilise un système de détection en 2 phases qui fonctionne sur n'importe quel site web, avec une précision optimisée pour les job boards connus. Ce système est capable d'identifier une offre d'emploi en quelques millisecondes, y compris sur des sites jamais rencontrés auparavant.

Phase 1 : Pré-détection rapide (~1 ms)

Trois vérifications structurelles ultra-rapides, exécutées en parallèle :

  1. Métadonnées structurées : Recherche du schéma JobPosting dans les données structurées de la page (standard Schema.org utilisé par les sites optimisés pour le référencement)
  2. Marqueurs de genre dans les titres : Détection de patterns typiques comme (H/F), (m/w/d), (F/H/X) dans les titres principaux, courants dans les offres d'emploi en 4 langues
  3. Segments d'URL : Analyse du chemin de l'URL pour détecter des termes comme /jobs/, /careers/, /emplois/, /stellenangebote/ en 6 langues

Si l'une de ces vérifications réussit, la page est immédiatement identifiée comme offre d'emploi sans passer à la Phase 2.

Phase 2 : Scoring heuristique (~5-10 ms)

Si la Phase 1 échoue, un système de scoring analyse le contenu de la page. Le seuil de détection est fixé à 40 points.

Critère Points Maximum Description
Bouton "Postuler" / "Apply" +30 30 Détection multilingue des boutons de candidature en français, anglais, allemand et espagnol, incluant les variantes idiomatiques (Je postule, Jetzt bewerben, Enviar candidatura)
Titres de sections métier +15 chacun 30 Titres typiques comme Missions, Requirements, Aufgaben, Profil recherché
Mots-clés emploi +2 chacun 20 Termes courants : CDI, full-time, Vollzeit, salario, télétravail
Contenu structuré +10 10 Zone de contenu principal dépassant 500 caractères
Détection universelle : Même sur un site jamais rencontré, l'extension peut identifier une offre d'emploi grâce à ce système de scoring multilingue. Elle n'est pas limitée aux 14+ sites connus.

Gestion des applications web modernes (SPA)

L'extension gère nativement les sites modernes construits avec des frameworks comme React ou Angular, où la navigation ne recharge pas la page :

  • Observation des changements d'URL : Détection automatique des navigations côté client sans rechargement de page
  • Observation du contenu : Surveillance des panneaux latéraux (utilisée par Indeed, LinkedIn, Glassdoor) qui changent le contenu affiché sans modifier l'URL
  • Réinjection automatique : Si un re-rendu du site supprime le bouton FitMyCV, celui-ci est réinjecté automatiquement avec un système de tentatives progressives

Extraction intelligente du contenu

Une fois l'offre détectée, l'extension extrait le contenu pertinent en éliminant automatiquement la navigation, les publicités et les éléments parasites. Elle utilise une double stratégie d'extraction pour garantir un résultat optimal sur tous les sites.

Stratégie 1 : Algorithme d'extraction du contenu principal

L'extension utilise un algorithme avancé d'extraction de contenu (le même que celui du mode lecture des navigateurs) qui isole automatiquement le texte pertinent de la page en éliminant la navigation, les publicités, les menus et tout élément superflu. Le contenu HTML est ensuite converti en texte structuré (titres, listes, paragraphes) prêt pour le traitement par l'IA.

Stratégie 2 : Sélecteurs ciblés par site

Pour chaque job board connu, l'extension dispose de règles d'extraction spécifiques qui ciblent directement la zone de description de l'offre. Cette approche garantit une précision maximale sur les sites les plus populaires, indépendamment de leur mise en page.

Les deux résultats sont comparés et le meilleur est conservé. Cette approche duale permet une extraction fiable aussi bien sur les sites connus que sur les sites découverts pour la première fois.

Scoring de qualité de l'extraction (0-100)

Chaque extraction est automatiquement évaluée sur 4 axes. Le contenu est considéré valide s'il atteint un score minimum de 25/100 et dépasse 200 caractères. Ce scoring alimente un indicateur visuel dans l'interface (badge vert, jaune ou rouge).

Composante Poids Critères
Longueur 0-20 pts Score optimal entre 1 500 et 8 000 caractères, dégradation progressive au-delà
Mots-clés pertinents 0-30 pts Présence de termes essentiels (mission, profil, compétences) et bonus (salaire, remote, CDI)
Structure 0-25 pts Présence de listes à puces, de titres et de paragraphes bien formés
Bruit (pénalité) 0 à -25 pts Contenu parasite détecté (mentions de cookies, politique de confidentialité, newsletter)

Déduplication des offres

Pour éviter les doublons, l'extension normalise les URLs des offres. Sur les grands sites d'emploi (LinkedIn, Indeed), des méthodes spécifiques extraient l'identifiant unique de l'offre, garantissant qu'une même offre n'est capturée qu'une seule fois, quel que soit le chemin de navigation emprunté.

Authentification JWT dédiée

L'extension utilise un système d'authentification indépendant du cookie de session web, conçu spécifiquement pour le contexte d'extension navigateur. Il repose sur un jeton JWT signé côté serveur, valide 7 jours.

Deux méthodes de connexion

  1. Email et mot de passe : Saisie directe dans l'interface de l'extension avec validation côté serveur
  2. OAuth (Google / GitHub) : L'extension ouvre un onglet vers le SaaS qui gère le flux de connexion standard, puis transmet le jeton de manière sécurisée à l'extension

Renouvellement intelligent du jeton

Mécanisme Déclencheur Description
Refresh proactif Vérification périodique (12h) + avant chaque requête Si le jeton expire dans moins de 24h, il est renouvelé automatiquement en arrière-plan, sans intervention de l'utilisateur
Refresh réactif Réception d'une réponse "non autorisé" Tentative automatique de renouvellement puis rejeu de la requête initiale
Expiration définitive Échec du renouvellement Nettoyage du jeton, arrêt de la surveillance, retour automatique à l'écran de connexion
Sécurité : Le jeton est signé côté serveur et vérifié à chaque requête par un middleware dédié. La transmission du jeton OAuth entre la page web et l'extension est sécurisée par validation d'origine.

Flux de données : de l'offre au CV

L'extension envoie le contenu extrait au serveur, qui prend ensuite le relais pour la structuration et la génération du CV. Ce flux permet de centraliser l'intelligence IA côté serveur tout en offrant une expérience fluide côté navigateur.

Flux de données Extension vers Pipeline de génération

Étapes détaillées

  1. L'extension extrait le contenu de l'offre et le convertit en texte structuré côté navigateur
  2. Le contenu est envoyé au serveur qui exécute une extraction IA pour structurer les données en format exploitable (titre, entreprise, compétences, etc.)
  3. Le pipeline standard de génération de CV prend le relais (le même que pour les offres soumises via URL ou PDF)
  4. L'extension suit la progression via une surveillance périodique : rafraîchissement toutes les 5 secondes quand l'interface est ouverte, toutes les 30 secondes en arrière-plan
  5. Le CV généré est accessible en un clic depuis l'extension, avec ouverture directe dans le SaaS et pré-sélection automatique du document

Interface de l'extension (Popup)

L'interface est une fenêtre compacte en mode sombre, conçue pour être efficace sans quitter la page en cours. Elle est disponible en 4 langues avec changement instantané.

Sélecteur de langue 4 drapeaux en pied de page, changement instantané de toute l'interface

Bouton "FitMyCV +" injecté dans les pages

Sur les job boards connus, un bouton est injecté directement dans la page à côté du titre de l'offre. Ce bouton :

  • S'adapte visuellement au design de chaque site (taille, espacement, disposition) pour une intégration naturelle
  • Présente 5 états visuels avec retour immédiat : par défaut, chargement, succès, doublon et erreur
  • Est traduit dynamiquement selon la langue choisie dans l'extension
  • Persiste automatiquement malgré les re-rendus des applications web modernes
