Extension Navigateur

Capturez et adaptez une offre d'emploi en un clic, sans quitter votre navigateur. L'extension FitMyCV pour Chrome et Firefox transforme votre navigation quotidienne en outil de productivité : elle detecte automatiquement les offres d'emploi sur plus de 14 sites, extrait leur contenu et lance la generation d'un CV optimise directement depuis la page.

Architecture de l'extension

L'extension repose sur l'architecture Manifest V3, le standard actuel impose par les navigateurs. Elle est construite autour de 5 composants specialises qui communiquent entre eux via un systeme de messages interne.

Compatibilite Chrome et Firefox

L'extension est developpee a partir d'une seule base de code avec une compatibilite native Chrome et Firefox. Le systeme de build produit automatiquement deux versions adaptees a chaque navigateur, sans duplication de code. L'internationalisation couvre 4 langues (francais, anglais, espagnol, allemand) avec detection automatique de la langue du navigateur.

Detection intelligente des offres d'emploi

L'extension utilise un systeme de detection en 2 phases qui fonctionne sur n'importe quel site web, avec une precision optimisee pour les job boards connus. Ce systeme est capable d'identifier une offre d'emploi en quelques millisecondes, y compris sur des sites jamais rencontres auparavant.

Phase 1 : Pre-detection rapide (~1 ms)

Trois verifications structurelles ultra-rapides, executees en parallele :

  1. Metadonnees structurees : Recherche du schema JobPosting dans les donnees structurees de la page (standard Schema.org utilise par les sites optimises pour le referencement)
  2. Marqueurs de genre dans les titres : Detection de patterns typiques comme (H/F), (m/w/d), (F/H/X) dans les titres principaux, courants dans les offres d'emploi en 4 langues
  3. Segments d'URL : Analyse du chemin de l'URL pour detecter des termes comme /jobs/, /careers/, /emplois/, /stellenangebote/ en 6 langues

Si l'une de ces verifications reussit, la page est immediatement identifiee comme offre d'emploi sans passer a la Phase 2.

Phase 2 : Scoring heuristique (~5-10 ms)

Si la Phase 1 echoue, un systeme de scoring analyse le contenu de la page. Le seuil de detection est fixe a 40 points.

Critere Points Maximum Description
Bouton "Postuler" / "Apply" +30 30 Detection multilingue des boutons de candidature en francais, anglais, allemand et espagnol, incluant les variantes idiomatiques (Je postule, Jetzt bewerben, Enviar candidatura)
Titres de sections metier +15 chacun 30 Titres typiques comme Missions, Requirements, Aufgaben, Profil recherche
Mots-cles emploi +2 chacun 20 Termes courants : CDI, full-time, Vollzeit, salario, teletravail
Contenu structure +10 10 Zone de contenu principal depassant 500 caracteres
Detection universelle : Meme sur un site jamais rencontre, l'extension peut identifier une offre d'emploi grace a ce systeme de scoring multilingue. Elle n'est pas limitee aux 14+ sites connus.

Gestion des applications web modernes (SPA)

L'extension gere nativement les sites modernes construits avec des frameworks comme React ou Angular, ou la navigation ne recharge pas la page :

  • Observation des changements d'URL : Detection automatique des navigations cote client sans rechargement de page
  • Observation du contenu : Surveillance des panneaux lateraux (utilisee par Indeed, LinkedIn, Glassdoor) qui changent le contenu affiche sans modifier l'URL
  • Reinjection automatique : Si un re-rendu du site supprime le bouton FitMyCV, celui-ci est reinjecte automatiquement avec un systeme de tentatives progressives

Extraction intelligente du contenu

Une fois l'offre detectee, l'extension extrait le contenu pertinent en eliminant automatiquement la navigation, les publicites et les elements parasites. Elle utilise une double strategie d'extraction pour garantir un resultat optimal sur tous les sites.

Strategie 1 : Algorithme d'extraction du contenu principal

L'extension utilise un algorithme avance d'extraction de contenu (le meme que celui du mode lecture des navigateurs) qui isole automatiquement le texte pertinent de la page en eliminant la navigation, les publicites, les menus et tout element superflu. Le contenu HTML est ensuite converti en texte structure (titres, listes, paragraphes) pret pour le traitement par l'IA.

Strategie 2 : Selecteurs cibles par site

Pour chaque job board connu, l'extension dispose de regles d'extraction specifiques qui ciblent directement la zone de description de l'offre. Cette approche garantit une precision maximale sur les sites les plus populaires, independamment de leur mise en page.

Les deux resultats sont compares et le meilleur est conserve. Cette approche duale permet une extraction fiable aussi bien sur les sites connus que sur les sites decouverts pour la premiere fois.

Scoring de qualite de l'extraction (0-100)

Chaque extraction est automatiquement evaluee sur 4 axes. Le contenu est considere valide s'il atteint un score minimum de 25/100 et depasse 200 caracteres. Ce scoring alimente un indicateur visuel dans l'interface (badge vert, jaune ou rouge).

Composante Poids Criteres
Longueur 0-20 pts Score optimal entre 1 500 et 8 000 caracteres, degradation progressive au-dela
Mots-cles pertinents 0-30 pts Presence de termes essentiels (mission, profil, competences) et bonus (salaire, remote, CDI)
Structure 0-25 pts Presence de listes a puces, de titres et de paragraphes bien formes
Bruit (penalite) 0 a -25 pts Contenu parasite detecte (mentions de cookies, politique de confidentialite, newsletter)

Deduplication des offres

Pour eviter les doublons, l'extension normalise les URLs des offres. Sur les grands sites d'emploi (LinkedIn, Indeed), des methodes specifiques extraient l'identifiant unique de l'offre, garantissant qu'une meme offre n'est capturee qu'une seule fois, quel que soit le chemin de navigation emprunte.

Authentification JWT dediee

L'extension utilise un systeme d'authentification independant du cookie de session web, concu specifiquement pour le contexte d'extension navigateur. Il repose sur un jeton JWT signe cote serveur, valide 7 jours.

Deux methodes de connexion

  1. Email et mot de passe : Saisie directe dans l'interface de l'extension avec validation cote serveur
  2. OAuth (Google / GitHub) : L'extension ouvre un onglet vers le SaaS qui gere le flux de connexion standard, puis transmet le jeton de maniere securisee a l'extension

Renouvellement intelligent du jeton

Mecanisme Declencheur Description
Refresh proactif Verification periodique (12h) + avant chaque requete Si le jeton expire dans moins de 24h, il est renouvele automatiquement en arriere-plan, sans intervention de l'utilisateur
Refresh reactif Reception d'une reponse "non autorise" Tentative automatique de renouvellement puis rejeu de la requete initiale
Expiration definitive Echec du renouvellement Nettoyage du jeton, arret de la surveillance, retour automatique a l'ecran de connexion
Securite : Le jeton est signe cote serveur et verifie a chaque requete par un middleware dedie. La transmission du jeton OAuth entre la page web et l'extension est securisee par validation d'origine.

Flux de donnees : de l'offre au CV

L'extension envoie le contenu extrait au serveur, qui prend ensuite le relais pour la structuration et la generation du CV. Ce flux permet de centraliser l'intelligence IA cote serveur tout en offrant une experience fluide cote navigateur.

Flux de donnees Extension vers Pipeline de generation
sequenceDiagram participant U as Utilisateur participant EXT as Extension participant SRV as Serveur FitMyCV participant IA as Pipeline IA U->>EXT: Capture l offre (clic bouton) EXT->>EXT: Extraction du contenu en texte structure EXT->>EXT: Scoring qualite (0-100) U->>EXT: Selectionne un CV source + lance la generation EXT->>SRV: Envoie le contenu structure de l offre SRV->>SRV: Extraction IA des donnees structurees (JSON) SRV->>IA: Lance le pipeline de generation de CV IA-->>SRV: CV adapte genere SRV-->>EXT: Progression en temps reel (polling) EXT-->>U: Suivi de la progression dans l interface U->>EXT: Clic sur le CV termine EXT->>SRV: Ouvre le CV dans le SaaS

Etapes detaillees

  1. L'extension extrait le contenu de l'offre et le convertit en texte structure cote navigateur
  2. Le contenu est envoye au serveur qui execute une extraction IA pour structurer les donnees en format exploitable (titre, entreprise, competences, etc.)
  3. Le pipeline standard de generation de CV prend le relais (le meme que pour les offres soumises via URL ou PDF)
  4. L'extension suit la progression via une surveillance periodique : rafraichissement toutes les 5 secondes quand l'interface est ouverte, toutes les 30 secondes en arriere-plan
  5. Le CV genere est accessible en un clic depuis l'extension, avec ouverture directe dans le SaaS et pre-selection automatique du document

L'interface est une fenetre compacte en mode sombre, concue pour etre efficace sans quitter la page en cours. Elle est disponible en 4 langues avec changement instantane.

Bouton "FitMyCV +" injecte dans les pages

Sur les job boards connus, un bouton est injecte directement dans la page a cote du titre de l'offre. Ce bouton :

  • S'adapte visuellement au design de chaque site (taille, espacement, disposition) pour une integration naturelle
  • Presente 5 etats visuels avec retour immediat : par defaut, chargement, succes, doublon et erreur
  • Est traduit dynamiquement selon la langue choisie dans l'extension
  • Persiste automatiquement malgre les re-rendus des applications web modernes
